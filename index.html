關於 MDX 字典

    

      MDX 是 MDict 格式的字典檔，可從網路免費取得大量英漢字典。

      建議字典：牛津英漢詞典、朗文英漢詞典、Collins English Dictionary 等。

      字典資料存於瀏覽器 IndexedDB，不會上傳至任何伺服器。

      查詢順序：字典 → AI 補充缺漏欄位

☁️ Google Drive 設定說明

      
    

    

      
📋 步驟說明

      
1. 前往 Google Cloud Console ↗

      
2. 建立新專案（或選現有專案）

      
3. 左側選單 → API 和服務 → 程式庫，搜尋「Google Drive API」並啟用

      
4. 左側選單 → API 和服務 → 憑證

      
5. 點「建立憑證」→ OAuth 用戶端 ID

      
6. 應用程式類型選 Web 應用程式

      
7. 「已授權的 JavaScript 來源」加入你使用的網址，例如：

          • https://你的帳號.github.io

          • 本地測試加 http://localhost 和 null
      

      
8. 建立後複製「用戶端 ID」貼到設定欄位

      

        ⚠️ 本機開啟 HTML 時（file://），OAuth 可能受限。建議部署到 GitHub Pages 再使用。
